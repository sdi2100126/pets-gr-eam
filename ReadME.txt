ΥΛΟΠΟΙΗΣΗ:
Υλοποιήθηκε πλήρως η διεπαφή του ιστοχώρου Pets.gr με React.

Λειτουργίες που υλοποιήθηκαν:

ΓΙΑ ΙΔΙΟΚΤΗΤΕΣ:
- Προβολή και εκτύπωση βιβλιαρίου υγείας κατοικιδίου
- Δήλωση απώλειας κατοικιδίου (με προσωρινή/οριστική αποθήκευση)
- Δήλωση εύρεσης κατοικιδίου
- Ιστορικό δηλώσεων
- Αναζήτηση κτηνιάτρων με φίλτρα (περιοχή, ειδικότητα, εμπειρία, διαθεσιμότητα)
- Προβολή προφίλ κτηνιάτρου
- Προγραμματισμός ραντεβού (5-steps)
-firestore με εξαίρεση τον προγραμματισμό ραντεβού που δεν πρόλαβε να ολοκληρωθεί


ΓΙΑ ΚΤΗΝΙΑΤΡΟΥΣ:
- Δημιουργία και επεξεργασία δημόσιου προφίλ
- Καταγραφή ταυτότητας κατοικιδίου
- Καταγραφή συμβάντων ζωής (απώλεια, εύρεση, μεταβίβαση, υιοθεσία)
- Καταγραφή ιατρικών πράξεων
- Ιστορικό επισκέψεων και ιατρικών πράξεων
- Διαθεσιμότητα (ημέρες και ώρες)
- Παρακολούθηση και διαχείριση ραντεβού
- Προβολή αξιολογήσεων
-state variables

ΓΙΑ ΟΛΟΥΣ:
- Αναζήτηση απολεσθέντων κατοικιδίων (χωρίς login)
- Αναφορά εύρεσης κατοικιδίου (από πολίτες)
- Αναζήτηση κτηνιάτρων (χωρίς login)
- Πληροφορίες για ιδιοκτήτες και κτηνιάτρους
-state variables
